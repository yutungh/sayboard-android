# Security

This project is an Android keyboard prototype. Keyboards are sensitive software because they can receive typed text, record microphone audio when granted permission, and insert text into other apps.

Do not hardcode API keys. Use the in-app settings for local development, and migrate to encrypted preferences or Android Keystore before any production release.

If you find a security issue, please open a private security advisory on GitHub if available for the repository owner, or contact the maintainer directly.
